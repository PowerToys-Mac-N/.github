## ЁЯУД Overview

<a href="#powerToys">PowerToys</a> is Microsoft's collection of power user utilities for Mac тАФ a suite of focused tools that fill gaps in the standard macOS experience, extend existing system capabilities, and add productivity features that demanding Mac users have historically found missing from the operating system.

The <a href="#powerToys">FancyZones window management</a> lets users define custom window layout zones on the screen тАФ grid layouts, side-by-side arrangements, and complex multi-zone configurations that windows snap to when dragged. Rather than the binary split-screen of macOS's native Split View, FancyZones enables precise tiling layouts that match individual workflow patterns тАФ a code editor on the left third, a browser in the center, and documentation on the right third, maintained as a snappable layout. The <a href="#powerToys">Color Picker</a> captures the exact color value of any pixel on screen from a system-wide keyboard shortcut тАФ displaying hex, RGB, HSL, and other color formats and copying the value to clipboard for use in design tools and code without switching to a dedicated color picker application.

The <a href="#powerToys">Peek file preview</a> enables Quick Look-style preview of files from the keyboard тАФ previewing images, documents, and other files without fully opening them, providing faster file review in workflows where many files need to be evaluated quickly. The <a href="#powerToys">keyboard remapping</a> reassigns individual keys and keyboard shortcuts system-wide тАФ remapping keys that clash with Mac conventions for Windows users transitioning to Mac, creating custom shortcuts for applications, and reconfiguring the keyboard for ergonomic preferences without third-party keyboard management software.

<a href="#powerToys">PowerRename bulk renaming</a> applies find-and-replace, regex patterns, and naming conventions to groups of files simultaneously тАФ renaming hundreds of files according to a consistent convention without manual individual renaming. <a href="#powerToys">Text Extractor</a> uses OCR to extract text from any part of the screen тАФ copying text from images, screenshots, PDFs displayed on screen, and any visual content where the text isn't selectable with standard methods.

<a href="#powerToys">Always on Top</a> pins any window to remain above all other windows тАФ keeping a reference document, a communication window, or a timer visible while working in other applications. <a href="#powerToys">Image Resizer</a> right-click resizes images in Finder to predefined or custom dimensions тАФ batch resizing images without opening an image editor for the common workflow of resizing for email, web, or specific size requirements.
